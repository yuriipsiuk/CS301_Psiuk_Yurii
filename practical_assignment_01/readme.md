Репозиторій містить розв'язання першої практичної роботи з CS301, виконаної Юрієм Псюком - практична група 4.

Структура репозиторію:
1. practical_assignment_01.sql - скрпит зі схемою БД і головним запитом
2. *.csv - дані для імпорту в таблиці, що використовувалися в запиті

У БД міститься інформації про клієнтів, їх рахунки, транзакції, ресторани та історія замовлень за травень і червень.

Головний запит
міститься у скрипті після структури БД.
1. Об'єднує через UNION ALL таблиці замовлень за травень і червень за допомогою CTE
2. Робить JOIN 5 таблиць для повної інформації про клієнтів
3. Фільтрує дані за містом і групує їх, щоб можна було порахувати окремо для кожного клієнта витрати
